Помощь

Это приложение для ведения онлайн-журнала оценки.

Загрузка журнала - на этой вкладке пользователь может загрузить таблицу формата .csv, .txt, .xlsx.

Редактирование журнала - Вкладка редактирования журнала, создания нового.
Для удаления записи об ученике нужно оставить строку пустой.

Таблица оценок - Представвление оценок в табличном виде.

График оценок - Статистик по журналу в графическом виде.
